# Prompts do Comanda Flow

Esta pasta guarda os prompts e especificacoes que guiam a evolucao do produto.

- `comanda-flow-master-prompt.md`: especificacao mestre de arquitetura, produto e desenvolvimento.
- `comanda-flow-relatorios-pdf.md`: especificacao focada no gerador de relatorios PDF profissional.
- `comanda-flow-final-prompt.md`: versao consolidada que une arquitetura geral e padrao premium de relatorios PDF.

Use `comanda-flow-final-prompt.md` como fonte de verdade principal antes de criar novos modulos, refatorar arquitetura ou implementar funcionalidades grandes.
